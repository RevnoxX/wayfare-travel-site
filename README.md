# 🌍 Wayfare – Travel Booking & Exploration Platform

Wayfare is a modern travel booking and exploration platform built using **HTML5, CSS3, Vanilla JavaScript, Vite, and Supabase**. It enables users to explore travel destinations, manage bookings, maintain wishlists, and access a personalized dashboard, while administrators can efficiently manage packages, bookings, users, and enquiries through a dedicated admin panel.

---

## ✨ Features

### 👤 User Features
- Secure Login System
- Explore Travel Packages
- Book Tour Packages
- User Dashboard
- Wishlist Management
- Booking History
- Profile Management
- Reviews & Ratings
- Notifications
- Contact & Enquiry Form

### 🛠️ Admin Features
- Admin Authentication
- Dashboard Analytics
- Package Management
- Booking Management
- User Management
- Enquiry Management
